About the project and website

For the past two weeks or so I have been building this section of my website to represent what my life could possibly look like after college. This included creating eight different pages that cover the possibilities of my financial life, job, housing, etc. I did this for my Economics and Personal Finance final project, but it has been a real adventure getting to re-explore the intricacies of HTML and CSS (even though my code is a mess). I probably wont end up using JavaScript at all for this project, but I will likely use it in other parts of this website. One key thing to be aware of before continuing is that we were granted a $10,000 allowance based on the assumption that we would save and recieve graduation gifts.
