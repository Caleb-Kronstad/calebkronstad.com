NIFTY ENGINE

Nifty Engine is a work-in-progress 3D game engine developed in C++ with OpenGL


Features In-Progress

Features that are currently being implemented into Nifty Engine

  • Skinned meshes
  • Skeletal animation system
  • API improvements
  • User interface improvements

Current Features

Features that can be found in Nifty Engine

  • Basic physics
  • Basic collisions
  • Lighting
  • Shadows
  • Shader control
  • File control
  • Basic user interface (entity explorer, properties page, file manager, etc.)
